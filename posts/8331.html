MySQL案例4 MySQL实验(三)数据库更新操作

考核内容及实验目的：

使用SQL语句进行更新操作。

一、实验描述

导入一个外部数据库文件(world.sql)到Navicat图形化管理软件中,并进行进行单表查询、多表查询及嵌套查询操作。数据库world包含了country、countrylanguag、city三张表，涵盖了全世界的国家、国家语言和城市的相关信息。

下图是world数据库中的部分表信息

  • country表

  • countrylanguage表

  • city表

1、更新操作

（1）将Taiwan名下的所有城市的countryCode改为CHN，省名改为Taiwan。（提示：一次性完成更改用update 表名 set 操作1，操作2，… … where 条件）

（2）将country中香港（Hong Kong）、澳门（Macao）改为中国香港、中国澳门相关信息也进行更改。

（3）将countrylangue表中所有countryCode为TWN的变为CHN，并且将官方语言变为非官方，并且改变语言使用人口比例（假设台湾人口占中国人口的1.3%）。

2、删除操作

（1）country表中的name为Taiwan的元组。

（2）删除人口为68000的国家下设的所有城市。（参照例题，只删除城市）

3、插入操作

（1）创建一张表check_test1，一次性插入我国下辖的所有省和省总人口。（提示：例3.72）

（2）创建一张表check_test2，一次性插入官方语言为Chinese的国家（除中国）的所有城市（提示用连接）。

二、上机操作

0、实验环境

  • 硬件平台：一台笔电，操作系统为Win10pro 64位

  • 软件环境：

    数据库Server版本，8.0.30 MySQL Community Server

    图形化管理工具，Navicat Premium 15.0.25
